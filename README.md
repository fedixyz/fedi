# Fedi Monorepo

This repo contains all source code for the Fedi tech stack:

## Directory Structure

- /.github
  - workflows for CICD
- /bridge
  - contains Rust code used by the mobile app
- /docs
- /fedi-social-client
- /fedi-social-common
- /fedi-social-server
- /fedimintd
- /misc
- /ui
  - contains React PWA & React Native code for apps
